# 💖 Deep Connection Toolkit

Discover how deeply you connect with someone special. Analyze compatibility across movies, music, books, food, hobbies, and core values.

## ✨ Features

- **Multi-Category Analysis** - Movies, music, books, food, hobbies, values
- **Real-Time Compatibility Scoring** - 0-100% with smart algorithm
- **AI-Powered Insights** - Personalized observations about your connection
- **Beautiful Visualizations** - Progress bars, gradients, and animations
- **Shared Passions Discovery** - See exactly where you align

## 🚀 Quick Start
```bash
# Clone the repo
git clone https://github.com/EnergizedAstronaut/Deep-Connection-Toolkit.git
cd Deep-Connection-Toolkit

# Install dependencies
npm install

# Run locally
npm run dev
```

Visit `http://localhost:3000`

## 🎯 How It Works

1. **Enter Names** - Add names for both people
2. **Select Category** - Choose movies, music, books, etc.
3. **Pick Items** - Each person selects their favorites
4. **See Compatibility** - Watch the score calculate in real-time
5. **Read Insights** - Get personalized connection analysis

## 🎨 Categories

- 🎬 **Movies** - Discover shared film tastes
- 🎵 **Music** - Find musical common ground
